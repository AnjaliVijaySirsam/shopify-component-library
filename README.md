# shopify-component-library
Reusable Shopify OS 2.0 sections, snippets, and patterns built from real production projects.


## Featured Collection Section

Displays products from a selected collection with configurable layout options.

### Features
- Collection picker
- Product limit control
- Responsive grid settings
- Optional vendor and price display

### Shopify compatibility
- Online Store 2.0

### Notes
- Designed for reuse across multiple stores
- No hardcoded product handles
- Safe fallback when collection is empty
